Tutorial que demonstra o uso do sensor térmico MLX90614, que na verdade foi utilizado um módulo I2C com 2 sensores, sendo um o próprio, que faz leitura térmica de forma ótica, sem contato e outro na placa, que permite medições diretas na própria placa. Além disso foi utilizado um display LCD 16X2 com um módulo I2C, muito mais prático de usar que ligando o display diretamente no arduino, permitindo usar apenas a porta I2C, com 2 pinos, disponibilizando todo o resto.

Espero que este tutorial sirva de auxílio para aqueles que estejam enfrentando problemas com o uso desses módulos ou que precisem de soluções para medição de temperatura à distância e também usar displays sem usar muitas portas.
